---
title: 数组去重
---

## 利用 filter + indexOf 去重

```js
function removeDuplicate(arr) {
  return arr.filter((item, index, array) => array.indexOf(item) === index);
}
```

## 利用 reduce + includes 去重

```js
function removeDuplicate(arr) {
  return arr.reduce(
    (prev, cur) => (prev.includes(cur) ? prev : [...prev, cur], [])
  );
}
```

## 利用 Set 数据结构去重

```js
function removeDuplicate(arr) {
  return Array.from(new Set(arr));
}
```
