给你二叉树的根节点 root ，返回其节点值的 层序遍历 。 （即逐层地，从左到右访问所有节点）。

示例 1：

![sample_1](/img/code_leetcode_No.102_sample_1.png)

```md
输入：root = [3,9,20,null,null,15,7]
输出：[[3],[9,20],[15,7]]
```

示例 2：

```md
输入：root = [1]
输出：[[1]]
```

示例 3：

```md
输入：root = []
输出：[]
```

提示：

- 树中节点数目在范围 \[0, 2000\] 内
- -1000 <= Node.val <= 1000

## 实现

### 深度优先搜索 - bfs

```js
/**
 * @param {TreeNode} root
 * @return {number[][]}
 */
var levelOrder = function (root) {
  if (!root) return [];
  let res = [];
  let queue = [root];
  while (queue.length) {
    let len = queue.length;
    let level = [];
    for (let i = 0; i < len; i++) {
      let node = queue.shift();
      level.push(node.val);
      if (node.left) queue.push(node.left);
      if (node.right) queue.push(node.right);
    }
    res.push(level);
  }
  return res;
};
```
